폰트 적용 방법

Unity UI Toolkit을 사용하여 UI를 구성할 때, 폰트를 적용하는 방식에는 두 가지 방법이 있습니다. 하나는 폰트 파일을 직접 적용하는 방법이고, 다른 하나는 Unity에서 제공하는 폰트 애셋(Font Asset)을 사용하는 방법입니다.


처음 UI Builder 화면을 띄워서 기본 설정을 보면 Font Asset이 적요되어 있는 것을 볼 수 있습니다.

폰트 파일 적용시 주의


이런식으로 폰트 파일만 수정하게 되면 제대로 폰트가 적용되지 않습니다.
폰트가 제대로 적용되려면 위 두 가지 방법 중 하나만 선택하여 사용해야 하며, 혼용하게 되면 폰트 적용이 예상대로 이루어지지 않을 수 있습니다.


폰트 파일을 적용하려면 기존에 설정된 폰트 애셋을 None으로 처리해야 충돌이 발생하지 않습니다.
폰트 파일을 나중에 적용한 경우, 폰트 애셋은 더 이상 필요하지 않기 때문에 다음과 같이 None으로 설정해야 합니다.
