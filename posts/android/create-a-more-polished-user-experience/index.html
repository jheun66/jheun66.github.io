Material Components

UI/UX 측면에서 Material Components를 사용하는게 좋음

dependencies {
    ...
    implementation 'com.google.android.material:material:<version>'
}

Material Design Components Library를 사용하기 위해서 필요. 안드로이드 스튜디오 4.1 이상부터는 프로젝트를 생성할 때 자동으로 포함

Vector drawables backward compatibility solution

vectorDrawable은 API 21 이상부터 사용가능하기에 minSDK가 19이상이면 과거 안드로이드 버전에서도 지원가능하도록 해야함

app/build.gradle

android {
  defaultConfig {
    ...
    vectorDrawables.useSupportLibrary = true
   }
   ...
}

아니면 PNG 파일을 직접 만들기

Styles and Themes

  • res > values 디렉토리에 styles.xml 이름으로 Values Resource File 생성
<?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Widget.TipTime.TextView" parent="Widget.MaterialComponents.TextView">
        <item name="android:minHeight">48dp</item>
        <item name="android:gravity">center_vertical</item>
        <item name="android:textAppearance">?attr/textAppearanceBody1</item>
    </style>
</resources>
  • 컴포넌트를 스타일링할 때, 일반적으로 사용하고 있는 위젯을 확장
  • 새로운 스타일 이름은 자신의 앱 이름으로 치환하기
  • ?attr/textAppearanceBody1(Typography Theming 참고)
<TextView
    android:id="@+id/service_question"
    style="@style/Widget.TipTime.TextView"
    ... />

<TextView
    android:id="@+id/tip_result"
    style="@style/Widget.TipTime.TextView"
    ... />

style과 layout에 동일한 속성에 다른 값을 넣어줄 경우 layout file의 값 적용

  • 자주 사용하는 속성값들은 dimens.xml을 만들고 저장
<resources>
   <dimen name="min_text_height">48dp</dimen>
</resources>

Add these styles to your theme

위에처럼 직접 뷰에 넣어주는 방법 말고도 현재 테마에 포함하여 모든 해당 뷰에 적용 가능

theme.xml

<item name="textInputStyle">@style/Widget.MaterialComponents.TextInputLayout.OutlinedBox</item>
<item name="radioButtonStyle">@style/Widget.TipTime.CompoundButton.RadioButton</item>
<item name="switchStyle">@style/Widget.TipTime.CompoundButton.Switch</item>

가로 회전할 경우 화면을 넘어감

ScrollView로 ConstraintLayout을 감싸주기

엔터를 누를 경우 키보드 치워주기

override fun onCreate(savedInstanceState: Bundle?) {
    ...
    binding.costOfServiceEditText.setOnKeyListener { view, keyCode, _ -> handleKeyEvent(view, keyCode)
}

private fun handleKeyEvent(view: View, keyCode: Int): Boolean {
   if (keyCode == KeyEvent.KEYCODE_ENTER) {
       // Hide the keyboard
       val inputMethodManager =
           getSystemService(Context.INPUT_METHOD_SERVICE) as InputMethodManager
       inputMethodManager.hideSoftInputFromWindow(view.windowToken, 0)
       return true
   }
   return false
}
