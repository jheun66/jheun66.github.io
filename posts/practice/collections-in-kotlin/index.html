Learn about lambdas and higher-order functions

함수의 이름이 없이 중괄호로 함수의 몸통만 있는 표현식을 익명함수, 람다라 부름, 또한 코틀린에서는 함수를 매개변수, 리턴값으로도 사용할 수 있음

Function Type

fun main() {
    val triple: (Int) -> Int = { a: Int -> a * 3 }
    println(triple(5))
}
// 결과 15

Kotlin에서는 함수를 변수처럼 다룰 수 있는데 마찬가지로 타입을 가지게됨

(Int)→Int는 매개변수로 Int를 가지고 리턴값이 Int라는 의미

암시적으로 가지고 있는 매개변수 it이 있음, 생략가능

Higher-order functions

Kotlin에서는 함수를 매개변수, 리턴값으로도 사용가능, 이를 고차 함수라 부름

map, filter, foreach등도 고차함수의 예시

마찬가지로 sorteedWith에서도 람다함수를 넘겨줄 수 있음

fun main() {
    val peopleNames = listOf("Fred", "Ann", "Barbara", "Joe")

    println(peopleNames.sorted())
    // 길이로 정렬
    println(peopleNames.sortedWith { str1: String, str2: String -> str1.length - str2.length })
}

// 결과
//[Ann, Barbara, Fred, Joe]
//[Ann, Joe, Fred, Barbara]

OnClickListener and OnKeyListener in Android

안드로이드에서 고차함수, 람다함수를 제일 자주 보게되는 데는 클릭리스너라 생각됨

calculateButton.setOnClickListener(View.OnClickListener {
            override fun onClick(view: View?) {
                    handleClick()
            }
})
calculateButton.setOnClickListener{view -> handleClick()}
// 사용하지 않는 view 생략 가능
calculateButton.setOnClickListener{handleClick()}

// 생략 가능한 매개변수는 _로 표현 가능
costOfServiceEditText.setOnKeyListener { view, keyCode, _ -> handleKeyEvent(view, keyCode) }
